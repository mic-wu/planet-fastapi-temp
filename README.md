# Planet Story Explorer

## Overview

The **Planet Story Explorer** is an interactive web portal. It transforms a static gallery of stories into a living, intelligent, and contextualized atlas of global change. The goal is to provide a dynamic resource that updates daily, enhanced with AI-generated captions and geospatial context.

## Features

-   **Interactive Web Portal**: A polished web page built with a modern framework like React.
-   **Automated Content Enrichment Pipeline**: A backend engine that daily ingests new stories and enriches them.
-   **API & Data Store**: A robust RESTful API built with Python and FastAPI to serve enriched data.

## Technical Stack

-   **Frontend**: React
-   **Backend**: Python with FastAPI
-   **AI/ML**: Visual Large Language Model (VLLM) and Gemini Deep Research
